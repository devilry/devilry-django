{% blocktranslate trimmed %} Devilry supports basic markdown formatting. {% endblocktranslate %}

{% trans "Headings" %}

{% blocktranslate trimmed %} To create a heading, add one or more # symbols before your heading text. The number of # symbols will determine the heading size. One # symbol is the largest heading, and six # symbols is the smallest heading. {% endblocktranslate %}

{% blocktranslate trimmed %} Example heading level 1: # Heading 1 {% endblocktranslate %}

{% blocktranslate trimmed %} Example heading level 2: # Heading 3 {% endblocktranslate %}

{% blocktranslate trimmed %} Example heading level 6: # Heading 6 {% endblocktranslate %}

{% trans "Styling text" %}

{% blocktranslate trimmed %} You can style simple words or sentences with inlined bold or italic styling. {% endblocktranslate %}

{% trans "Bold" %}

{% blocktranslate trimmed %} Add two * symbols before and after the text you want bold formatting for. {% endblocktranslate %}

{% blocktranslate trimmed %} Example: **your text** {% endblocktranslate %}

{% trans "Italic" %}

{% blocktranslate trimmed %} Add one * symbol before and after the text you want italic formatting for. {% endblocktranslate %}

{% blocktranslate trimmed %} Example: *your text* {% endblocktranslate %}

{% trans "Links" %}

{% blocktranslate trimmed %} To format links, add a starting bracket symbol and a closing bracket symbol encapsulating the link-text. Directly follow this with a starting parantheses symbol and a closing parantheses symbol encapsulating the actual link address. {% endblocktranslate %}

{% blocktranslate trimmed %} Example: [My link text](https://www.devilry.test) {% endblocktranslate %}

{% trans "Code" %}

{% blocktranslate trimmed %} Mark code within a sentence with single backticks. The text encapsulated by the backticks will be formatted inline. {% endblocktranslate %}

{% blocktranslate trimmed %} Example: Use this command `ls -a` to do something. {% endblocktranslate %}
