.. _6.5.0changelog:

############################
What is new in Devilry 6.5.0
############################


Fixes
#####
This list is not exhaustive and may change before the final release.
- Update the low level web framework to Django 5.2.
- Improve accessibility in the student and examiner interfaces.
- Improve error reporting for background tasks.
- Improve several translations.
- Include course name in comment email subjects and templates
- Improve markdown sanitization by adding blockquote support.
- Various bug fixes and performance improvements.
