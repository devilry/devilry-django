############################
What is new in Devilry 5.5.0
############################


Fixes and new features
######################

- Better support for code snippets.
- New feature: Support for LaTeX match (was supported in devilry in an older version, so not strictly a new feature).
- New feature: Code diff support in comment editor
- New feature: Option that allow examiners to self-assign assignments
- New feature: Prompt students the assignment guidelines
- New feature: Examiners should see active periods
- New feature: Username "merge into" support
- WCAG fix: Too low contrast for warning note
- WCAG fix: Feedback feed skips H2 element wcag
